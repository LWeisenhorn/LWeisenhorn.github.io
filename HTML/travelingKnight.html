Traveling Knight Problem

I first came up with the idea for the project when I discovered that a knight on a chessboard can move to any given square if given enough moves. I decided it would be interesting to write a program that would optimize the moves from one square on the board to another.

The Development

This project is still a work in progress for when I am not busy with classes or work. I am currently modeling all of the moves the knight can do across the chessboard as one large graph data structure and using breadth-first search to analyze the ideal route from one node to another. I also plan for the future to have a fully complete front end for this project and allow the user to select the start and end nodes.
